### Visual Specifications

#### KPI Cards
- **Size:** 300x200px (each card)
- **Components:**
  - KPI Name (header)
  - Current Value (large number)
  - Target Value (small text)
  - % to Target (small text)
  - Status Indicator (traffic light icon)
- **Formatting:**
  - Background: White with subtle border
  - Status colors: Green (#00B050), Yellow (#FFC000), Red (#FF0000)
  - Font: Segoe UI, 24pt for value, 12pt for labels

#### Slicers
- **Type:** Dropdown or list
- **Size:** 200x40px each
- **Position:** Top right, horizontal layout
- **Styling:** Clean, modern, consistent with theme
