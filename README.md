# FLPP Power BI Dashboard

Power BI dashboard project for FLPP (Financial Leadership Performance Program).

## Overview

This repository contains the Power BI dashboard files and related resources for the FLPP project.

## Project Structure

```
.
├── README.md
├── .gitignore
└── dashboards/          # Power BI dashboard files (.pbix)
└── data/                # Data sources and datasets
└── docs/                # Documentation and notes
```

## Getting Started

1. Clone this repository
2. Open Power BI Desktop
3. Load the dashboard files from the `dashboards/` directory

## Requirements

- Power BI Desktop (latest version recommended)
- Access to required data sources

## Contributing

Please ensure all Power BI files are saved in the `dashboards/` directory before committing.
